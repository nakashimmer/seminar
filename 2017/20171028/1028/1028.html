W3C「PROPOSAL TO REPUBLISH PREVIOUS VERSIONS OF HTML AND XHTML AS OBSOLETE RECOMMENDATIONS」

we recommend that all implementers use the latest HTML5 –where HTML5 is the term to refer to the family of specifications. We are proposing to obsolete the particular 28 October 2014 W3C Recommendation which henceforth shall be called HTML 5.0, which has been superseded by HTML 5.1, a W3C Recommendation since 1 November 2016. This aims at clarifying the status of the document and point developers and implementers to the latest version of HTML5.

これからの展望として、2014年の10月末に W3Cが発表したHTML 5.0は、既に新しいのも出ててそちらが使われてることから、システムとして古く、この先それを使っての開発するとかあり得ない。だから2016年の11月にスタートしたHTML 5.1をこれからは標準にしましょう
