① [HTML] dialog要素

ダイアログ

HTML5はHTML,CSS,JavaScriptなどの仕様の総称名です。

HTML

<button id="btn1">ダイアログを開く</button>

<dialog id="dlg">
	<h3>ダイアログ</h3>
	<p>HTML5はHTML,CSS,JavaScriptなどの仕様の総称名です。</p>
	<p><button id="btn2">ダイアログを閉じる</button></p>
</dialog>

JavaScript

const DLG = document.getElementById("dlg");
const BTN1 = document.getElementById("btn1");
const BTN2 = document.getElementById("btn2");

BTN1.addEventListener("click", ()=>{
	DLG.showModal();
});

BTN2.addEventListener("click", ()=>{
	DLG.close();
});
