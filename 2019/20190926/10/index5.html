④ 音声でサーボモータ

次のどれかを大きな声でお願いします

  • 左上げて
  • 右上げて
  • 終了

JavaScript

 
RECOGNITION.addEventListener('result',(e)=>{
	result = e.results.item(i).item(0).transcript;
	if(result === "右上げて"){ SERVO.angle(0); }
	if(result === "左上げて"){ SERVO.angle(180); }
	if(result === "終了"){
		 SERVO.angle(90); RECOGNITION.stop(); 
	}
	i++;
});
