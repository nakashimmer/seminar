PWA(Progressive Web Apps)

PWA：デスクトップとモバイルの両方でインストール可能なアプリと同等の体験を提供し、ウェブ上で直接構築・配信される高速で信頼性の高いウェブアプリ

必要なもの

index.html 
navigator.serviceWorker.register(
	'./service-worker.js');
