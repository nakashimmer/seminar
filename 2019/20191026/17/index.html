dialog要素はダイアログボックスの要素

ダイアログ

HTML5はHTML,CSS,JavaScriptなどの仕様の総称名です。

HTML

 
<button id="btn1">ダイアログを開く</button>

<dialog id="dlg">
	<h3>ダイアログ</h3>
	<p>HTML5はHTML,CSS,JavaScriptなどの仕様の総称名です。</p>
	<p><button id="btn2">ダイアログを閉じる</button></p>
</dialog>

JavaScript

 
BTN1.addEventListener("click", ()=>{DLG.showModal();});
BTN2.addEventListener("click", ()=>{DLG.close();});
